# Вклад в Python Path

Проект пока ориентирован на одного пользователя, но улучшения приветствуются после выбора лицензии.

## Локальная разработка

```powershell
uv sync --extra dev
uv run uvicorn app.main:app --reload
```

## Перед pull request

```powershell
uv run ruff check app tests
uv run ruff format --check app tests
uv run pyright app tests
uv run pytest -q
```

## Правила для контента

- Один урок вводит один основной концепт и опирается на предыдущие.
- У каждого урока три теоретические карточки и три упражнения: выбор, короткий ответ, код.
- Кодовые задания должны быть короткими, детерминированными и совместимыми с учебным раннером.
- Не добавляйте большие заимствованные фрагменты чужих курсов. Пересказывайте идеи своими словами и ссылайтесь на первоисточник в PR.

## Коммиты и PR

- Делайте маленькие коммиты с действием в настоящем времени: `Добавить уроки по JSON`.
- В PR укажите пользовательский эффект, затронутые части программы и результаты проверок.
- Не включайте `python_path.db`, `.venv` и иные локальные артефакты.
