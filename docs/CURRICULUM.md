# Карта курса

Курс состоит из 120 микроуроков. Каждый раздел содержит три или четыре урока, затем контрольную точку. Новый материал открывается только после закрепления предыдущего — это сохраняет ощущение постепенного роста сложности.

| ---: | --- | --- |

| 2 | Логика и повторения | `if`, `for`, `while` |
| 3 | Функции и данные | Функции, списки, словари, множества |

| 5 | Операторы и выражения | Арифметика, boolean-логика, точность чисел |

## Принципы сложности

- Сначала один новый концепт за урок, затем — его комбинация с уже знакомыми.
- Кодовая практика опирается на конкретные тесты, а не на ручную проверку.
- Контрольная точка не открывается до завершения уроков раздела.
- Проекты в конце повторяют все ключевые слои: данные, логику, I/O, API, тесты и документацию.
